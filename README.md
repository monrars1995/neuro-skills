# Neuro Skills

Meta Ads automation skills for OpenCode/Claude Code.

## Skills

### 🎯 meta-ads-manager
Complete Meta Ads Traffic Manager with Graph API v21.0 integration. Handles account management, campaign creation, A/B testing, scaling, diagnostics, and reporting.

**Features:**
- Memory system with persistent account storage
- Campaign creation from briefing and creatives
- A/B testing framework
- Scaling playbook (horizontal/vertical)
- Performance diagnostics
- Industry benchmarks
- Automation rules
- Report templates

### 📋 traffic-strategist
Campaign preparation agent that analyzes folders, validates briefings, checks creatives, identifies gaps, and organizes files before campaign creation.

**Features:**
- Folder analysis
- Briefing validation
- Creative verification
- Gap identification
- Question generation for missing assets
- File organization
- Documentation generation (`analise.md`, `checklist.md`)

### ✍️ ad-copywriter
Ad copy specialist that analyzes brand voice, maintains tone consistency, creates on-brand copy for Feed/Stories/Reels/Carousel formats, and generates A/B testing variations.

**Features:**
- Brand voice analysis
- Tone consistency
- Copy for all Meta formats
- A/B testing variations
- Platform-specific optimization

## Workflow

```
traffic-strategist → ad-copywriter → meta-ads-manager
     (prep)            (copy)          (execute)
```

1. **traffic-strategist**: Analyze campaign folder, validate assets, identify gaps
2. **ad-copywriter**: Generate copy variations based on briefing and brand voice
3. **meta-ads-manager**: Create campaigns, launch ads, manage budgets

## Installation

Copy skills to your OpenCode skills directory:

```bash
cp -r meta-ads-manager ~/.opencode/skills/
cp -r traffic-strategist ~/.opencode/skills/
cp -r ad-copywriter ~/.opencode/skills/
```

## Usage

```
# Traffic Strategist
> Analyze the briefing in /campanhas/nike/2024-03/spring-sale/

# Ad Copywriter
> Create copy variations for the Nike Spring Sale campaign

# Meta Ads Manager
> List my saved ad accounts
> Create campaign from /campanhas/nike/2024-03/spring-sale/
```

## Directory Structure

```
/campanhas/{cliente}/{YYYY-MM}/{campanha}/
├── briefing.md
├── analise.md          # Generated by traffic-strategist
├── checklist.md       # Generated by traffic-strategist
├── copy_variants.md   # Generated by ad-copywriter
├── targeting.json     # Generated by ad-copywriter
├── brand_voice.json   # Generated by ad-copywriter
└── ad_*.*             # Creative assets
```

## Memory System

```
~/.meta-ads-manager/
├── accounts.json       # Saved ad accounts
├── session.json        # Current session data
├── cache/              # Insights cache
└── logs/               # Action history
```

## Requirements

- Meta Business Account
- Facebook App with Graph API v21.0
- Valid access token with ads_management permission
